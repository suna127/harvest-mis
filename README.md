# 🌾 農地 作付け計画ゲーム

ボロノイ図と最大重み独立集合問題を題材にした、農地の作付け計画をテーマにしたブラウザゲームです。

## ゲームの概要

> あなたは農場経営者。広大な農地に**管理小屋**を配置すると、各小屋が担当する**農地区画**（ボロノイ領域）が決まります。
> ただし**隣接する区画は同じ水路を共有**しているため、両方を同時に作付けすることができません。
> 今年の**収穫面積を最大化**する作付け計画を立てましょう！

## 遊び方

1. **農場配置フェーズ**: キャンバスをクリックして管理小屋を配置（最大40か所）
2. **作付け計画フェーズ**: 農地区画をクリックして作付け予定を選択。隣接する区画は同時作付け不可
3. **収穫結果フェーズ**: 選択した作付け計画が最適かどうかを判定

最大収穫面積を達成できればハイスコア更新！

## 技術的な詳細

| 項目 | 内容 |
|------|------|
| ボロノイ図生成 | 半平面交差法（Half-plane intersection） |
| 最適解計算 | 最大重み独立集合（MWIS）- 枝刈りバックトラッキング |
| フィールド規模 | 8km × 5km = 4,000ha |
| 管理小屋の最大数 | 40か所 |

### アルゴリズム

- **ボロノイ図**: 各セルをキャンバス矩形から全他母点との垂直二等分線で順次クリッピングして構築
- **MWIS**: ボロノイ隣接グラフは平面グラフ（平均次数≈6）のため、枝刈りバックトラッキングでN=40でも数ミリ秒以内に最適解を計算

## 実行方法

単一のHTMLファイルなので、ブラウザで `test.html` を開くだけで動作します。サーバーは不要です。

```
git clone https://github.com/suna127/harvest-mis.git
cd harvest-mis
# index.html をブラウザで開く
```
